Hyve: Note-Taking App

Project Overview

Hyve is an innovative note-taking platform designed for students and professionals to organize, share, and collaborate on academic notes and quizzes. Built with a modern tech stack, Hyve enhances learning experiences through interactive features and gamification. You can try accessing Hyve through this link: https://hyveapp.netlify.app/

Technologies Used

  • Frontend: React.js, TailwindCSS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: OAuth-based login
  • Hosting: Vercel (frontend), Heroku (backend)

Key Features

  • Collaborative note-sharing and quiz creation
  • Gamification with leaderboards and points
  • Secure OAuth-based authentication
  • Responsive design for seamless accessibility

Development Process

The development followed an agile methodology, with continuous feedback loops to enhance user experience. UI/UX was prototyped in Figma before implementation.

Project Outcome

Hyve successfully launched with positive feedback from users, enhancing note organization and collaborative learning. Future updates aim to introduce AI-powered summarization features.
